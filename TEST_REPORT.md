## 5. 关键用户流程验证

### 流程 1: 用户注册 → 登录
```
1. 用户访问首页
2. 点击登录/注册
3. 填写邮箱密码
4. 验证邮箱
5. 自动创建用户资料
6. 自动创建钱包
```

### 流程 2: 提交救援请求
```
1. 用户登录
2. 点击"提交救援申请"
3. 填写问题描述
4. 选择技术栈标签
5. 设置预算范围
6. 提交请求
7. AI 自动诊断
```

### 流程 3: 专家抢单
```
1. 专家登录
2. 浏览 Vibe Request 大厅
3. 查看请求详情
4. 提交抢单方案
5. 等待用户选择
6. 用户接受后创建订单
```

### 流程 4: 实时沟通
```
1. 订单创建后自动创建聊天室
2. 用户和专家可以实时聊天
3. 支持文字、代码、文件
4. WebSocket 实时推送
```
